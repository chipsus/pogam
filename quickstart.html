Getting Started¶

This tutorial will give you a quick tour of the Pogam scraping library. It is written for macOS but should work for Linux.

  • Requirements

  • Installation

  • First search

Requirements¶

Make sure you have the following installed:

  • Git

  • Pip

For dev work, we assume you have the following installed:

  • Make

Installation¶

The easiest way to install the library is by using pip:

$ pip install git+https://github.com/ludaavics/pogam.git

Verify your installation …

import pogam
print(pogam.__version__)

dev install¶

If you want to modify the code or contribute to the project you will want to clone the repository instead. The project ships with an initialization recipe to set up a conda environment and a git pre-commit hook:

$ git clone git+https://github.com/ludaavics/pogam.git
$ cd pogam
$ make init

If you don’t have make installed, you can initialize manually. From the project’s folder:

$ conda env create –file .ci/environment.yml $ ln -s ../../.ci/pre-commit .git/hooks/pre-commit
