Developers’ Guide¶

This document is for developers that want to modify or contribute to the project. It is written for macOS but should work for Linux.

  • Requirements

  • Installation

Requirements¶

In addition to the standard requirements, we assume you have the following installed:

  • Make

Note

For installation of make on Windows, we recommend using the Chocolatey package manager for Windows.

Installation¶

The project ships with an initialization recipe to set up a conda environment and a git pre-commit hook:

$ git clone git+https://github.com/ludaavics/pogam.git
$ cd pogam
$ make init

If you don’t have make installed, you can manually initialze the project. From the project’s folder:

$ conda env create --file .ci/environment.yml
$ ln -s ../../.ci/pre-commit .git/hooks/pre-commit
